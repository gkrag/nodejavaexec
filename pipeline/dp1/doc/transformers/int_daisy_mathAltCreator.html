Transformer Purpose

Provides text and image fallbacks to MathML islands in DTBook to support the MathML in DAISY extension.

Input Requirements

The input DTBook+MathML document.

Output

A DTBook+MathML document where MathML islands have text and image fallbacks.

Configuration/Customization

Parameters (tdf)

overwrite
Whether to overwrite existing MathML alttext and altimg attributes.

Dependencies

This transformer delegates its functionality to a third-party add-in, currently implemented by the Design Science MathDAISY product.

Author

MathDAISY by Design Science, Transformer wrapper by Markus Gylling.

Licensing

The transformer code in under LGPL, the Design Science MathDAISY code is commercial and closed source.
