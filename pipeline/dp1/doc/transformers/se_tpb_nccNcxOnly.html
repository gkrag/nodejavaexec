Transformer Purpose

Creates an NCC/NCX only fileset from a Daisy 2.02 or z3986-2005 fileset.

Input Requirements

Any valid Daisy 2.02 or z3986-2005 fileset should do.

Output

If the input is a Daisy 2.02 fileset, the output will be a NCC-only (i.e. no textual content document) Daisy 2.02 fileset.

If the input is a z3986-2005 fileset, the output will be a NCX-only (i.e. no DTBook document) z3986-2005 fileset.

On success

A valid Daisy 2.02 or z3986-2005 fileset is created.

On error

On error, this transformer will throw an exception and abort.

Configuration/Customization

Parameters (tdf)

manifest
The manifest file of the input fileset (i.e. the ncc.html of a Daisy 2.02 fileset, and the .opf file of a z3986-2005 fileset.
outDir
The directory to write the NCC/NCX-only fileset to.

Further development

Support for z3986-2005 filesets has not been implemented yet.

Dependencies

None.
