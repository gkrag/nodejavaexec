Transformer Purpose

Convert a DTBook document to a text-only DAISY 3 book.

Note: this transformer uses the XSLT dtbook2daisy3textonly.xsl placed in org/daisy/util/xml/xslt/stylesheets.

Input Requirements

A DTBook document, valid to the 2005-1 DTD or later. This transformer does not support DTBook documents with MathML islands.

Output

A text-only DAISY 3 DTB (Z39.86-2005). Note that the spine of the generated DTB will be composed of only one big SMIL document.

Configuration/Customization

Parameters (tdf)

input
The path to the input DTBook file.
output
The path to the OPF manifest of the output DAISY 3 book.
copyReferring
Whether to copy auxiliary files (e.g. images) to the output directory.

Extended configurability

Further development

  • Add support for DTBook+MathML.
  • Add support for multiple SMIL spine items.

Dependencies

None. This transformer is a wrapper for an XSLT included in the Pipeline utility stylesheets.
